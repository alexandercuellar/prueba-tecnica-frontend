# 🚀 Prueba Técnica - Frontend (Angular)

## 📌 Descripción

Este proyecto corresponde al desarrollo del frontend de una aplicación web, construido en Angular como parte de una prueba técnica.

La aplicación permite la gestión de estudiantes y materias, consumiendo servicios desde una API REST desarrollada en .NET.

---

## ⚙️ Tecnologías utilizadas

* Angular
* TypeScript
* HTML / CSS
* Bootstrap (si usaste)

---

## ▶️ Ejecución del proyecto

### 1. Instalar dependencias

```bash
npm install
```

### 2. Ejecutar aplicación

```bash
ng serve
```

### 3. Acceder en navegador

```bash
http://localhost:4200/
```

---

## 🔗 Conexión con backend

Asegúrate de que el backend esté en ejecución para el correcto funcionamiento de la aplicación.
